Equinumerosity

Big Picture: We need a way to compare set sizes that works even without numbers. The answer: two sets have the same size if we can pair every element of one with exactly one element of the other — leaving nothing unpaired on either side. This pairing is called a bijection.

The Core Idea

Definition — Equinumerous Sets

Two sets A and B are equinumerous (written A ~ B or |A| = |B|) if there exists a bijection f: A → B — a function that is both injective and surjective.

Also called: having the same cardinality, being in one-to-one correspondence, or equipotent.

💡 Why Bijections?

Imagine matching socks: if every left sock has exactly one right sock partner, and every right sock is matched — the two piles are the same size. You never needed to count. A bijection does exactly this: it's a perfect matching with nothing left over.

Examples — Finite Sets

Example 1 — Simple Finite Sets

Let A = {cat, dog, fish} and B = {1, 2, 3}.
Define f: cat↦1, dog↦2, fish↦3. This is a bijection, so A ~ B and |A| = |B| = 3.

Example 2 — Checking Equinumerosity

Is {a,b,c,d} ~ {1,2,3}? No bijection exists — any function from a 4-element set to a 3-element set must send two elements to the same output (pigeonhole principle). So these are NOT equinumerous.

Examples — Infinite Sets (Surprising!)

Example 3 — Even Numbers ~ Natural Numbers

Let E = {0,2,4,6,…} (even naturals) and ℕ = {0,1,2,3,…}.
Define f: ℕ → E by f(n) = 2n.
Injective: f(m) = f(n) ⟹ 2m = 2n ⟹ m = n ✅
Surjective: every even number 2k = f(k) ✅
So E ~ ℕ! Even though E "seems smaller," they have the same cardinality.

Example 4 — Integers ~ Natural Numbers

Define a bijection f: ℕ → ℤ:
f(0)=0, f(1)=1, f(2)=-1, f(3)=2, f(4)=-2, f(5)=3, …
In general: f(n) = n/2 if n is even, f(n) = -(n+1)/2 if n is odd.
This pairs every natural number with exactly one integer — so |ℕ| = |ℤ|!

Example 5 — Open Interval ~ All Real Numbers

The interval (0,1) and ℝ are equinumerous!
The bijection f(x) = tan(π(x - ½)) maps (0,1) → ℝ bijectively.
So a tiny interval "contains as many points" as the entire real line.

Properties of Equinumerosity

Equinumerosity is an Equivalence Relation
  • Reflexive: A ~ A — the identity map id_A is a bijection. ✅
  • Symmetric: A ~ B ⟹ B ~ A — if f: A→B is a bijection, then f⁻¹: B→A is also a bijection. ✅
  • Transitive: A ~ B and B ~ C ⟹ A ~ C — compose the two bijections. ✅
Schröder–Bernstein Theorem

If there exist injections f: A → B and g: B → A, then A ~ B.
This is a powerful tool — you don't need to find an explicit bijection, just two injections!

⚠️ Common Misconception

A proper subset of an infinite set CAN be equinumerous to the whole set. In fact, this is Dedekind's definition of an infinite set: A set is infinite if and only if it is equinumerous to one of its proper subsets.

Cardinality of Sets & Countability

Big Picture: Cardinality is the "size" of a set. For finite sets, it's just the number of elements. For infinite sets, it's subtler — some infinities are larger than others! We classify sets as finite, countably infinite, or uncountably infinite.

Cardinal Numbers

Definition — Cardinality

The cardinality of a set A, written |A| or card(A), is a measure of its size.
Two sets have the same cardinality iff they are equinumerous (A ~ B).
For finite sets: |{a,b,c}| = 3. For infinite sets, we use special symbols.

Finite Sets

Definition — Finite Set

A set A is finite if A = ∅ or A ~ {1, 2, …, n} for some n ∈ ℕ.
Its cardinality is |A| = n (or 0 for the empty set).

Examples — Finite
  • |∅| = 0 — the empty set has zero elements.
  • |{red, green, blue}| = 3
  • |{x ∈ ℕ | x ≤ 100}| = 101
  • |𝒫({a,b})| = 4 — power set of a 2-element set has 4 subsets.

Countably Infinite Sets — ℵ₀

Definition — Countably Infinite

A set A is countably infinite if A ~ ℕ — there is a bijection between A and the natural numbers.
The cardinality is denoted ℵ₀ (read "aleph-naught" or "aleph-zero").
A set is countable if it is finite OR countably infinite.

Examples — Countably Infinite
  • ℕ = {0,1,2,3,…} — the natural numbers themselves. |ℕ| = ℵ₀
  • ℤ (integers) — bijection: 0,1,-1,2,-2,… |ℤ| = ℵ₀
  • Even numbers — bijection n↦2n. |E| = ℵ₀
  • ℚ (rationals) — Cantor's diagonal enumeration of fractions. |ℚ| = ℵ₀
  • Primes — infinitely many; can be listed p₁, p₂, p₃,… |Primes| = ℵ₀
  • ℕ × ℕ — all pairs of natural numbers; Cantor's diagonal shows this is countable.

Cantor's Diagonal Enumeration of ℚ

How to Count the Rationals

Arrange all positive fractions in a grid (rows = numerator, columns = denominator). Then zigzag diagonally through the grid, skipping fractions already seen. This produces a list 1/1, 1/2, 2/1, 3/1, 2/2, 1/3, 1/4, 2/3, 3/2, … that hits every rational number — proving ℚ is countable!

Uncountable Sets — 𝔠

Definition — Uncountably Infinite

A set is uncountable (or uncountably infinite) if it is infinite but NOT equinumerous to ℕ — no bijection with ℕ exists.
The cardinality of ℝ is denoted 𝔠 (the continuum) or 2^{ℵ₀}.
We have ℵ₀ < 𝔠 — the reals are a "bigger" infinity than the naturals.

Cantor's Diagonal Argument — ℝ is Uncountable

Suppose, for contradiction, that all real numbers in [0,1] can be listed: r₁, r₂, r₃, …
Write each in decimal form. Build a new number d by making its n-th decimal digit different from the n-th decimal digit of rₙ.
Then d ≠ rₙ for every n — so d is not on the list! Contradiction. Therefore ℝ is uncountable. □

Examples — Uncountable
  • ℝ (real numbers) — |ℝ| = 𝔠 = 2^{ℵ₀}
  • (0,1) — same cardinality as ℝ via f(x) = tan(π(x−½))
  • 𝒫(ℕ) — power set of ℕ is uncountable (Cantor's theorem)
  • ℝ² — the plane has the same cardinality as ℝ
  • Irrational numbers — uncountable (rationals countable, reals uncountable ⟹ irrationals uncountable)
⚠️ The Continuum Hypothesis

Is there a set with cardinality strictly between ℵ₀ and 𝔠? This is the Continuum Hypothesis. It was shown to be independent of standard set theory (ZFC) — meaning it can be neither proved nor disproved!

Cardinality of ℕ — the "smallest" infinite cardinal. Sets with |A|=ℵ₀ are countably infinite.
Is ℚ countable?
YES — Cantor's diagonal enumeration lists all rationals. |ℚ| = ℵ₀.

Arithmetic & Ordering of Cardinal Numbers

Big Picture: Cardinal numbers can be added, multiplied, and exponentiated — but the rules are very different from ordinary arithmetic, especially for infinite cardinals. Most strikingly: ℵ₀ + ℵ₀ = ℵ₀ and ℵ₀ × ℵ₀ = ℵ₀!

Ordering Cardinal Numbers

Definition — Cardinal Ordering

|A| ≤ |B| if there exists an injection f: A → B.
|A| < |B| if |A| ≤ |B| but A and B are NOT equinumerous.
By Schröder–Bernstein: if |A| ≤ |B| and |B| ≤ |A| then |A| = |B|.

Examples of Cardinal Ordering
  • 0 < 1 < 2 < … < n < … < ℵ₀ < 𝔠
  • |ℕ| < |ℝ| — the reals are a strictly larger infinity than the naturals.
  • |A| < |𝒫(A)| for any set A — power sets are always strictly larger (Cantor's theorem).
  • So: ℵ₀ < 2^{ℵ₀} < 2^{2^{ℵ₀}} < … — an endless tower of infinities!
Cantor's Theorem

For any set A: |A| < |𝒫(A)|.
There is no largest cardinal number — for every cardinal, there is a strictly larger one.

Cardinal Addition

Definition — Cardinal Addition

For disjoint sets A and B:
|A| + |B| = |A ∪ B|
(If not disjoint, use disjoint copies: A×{0} and B×{1}.)

Rules for Cardinal Addition
  • m + n = ordinary sum for finite cardinals.
  • ℵ₀ + n = ℵ₀ for any finite n — adding finitely many to infinity changes nothing.
  • ℵ₀ + ℵ₀ = ℵ₀ — two countable infinities merge into one (interleave them!).
  • 𝔠 + ℵ₀ = 𝔠 — adding countably many to an uncountable set doesn't grow it.
  • 𝔠 + 𝔠 = 𝔠 — two copies of ℝ have the same cardinality as one.
  • For infinite κ: κ + κ = κ (assuming Choice).
Example — ℵ₀ + ℵ₀ = ℵ₀

Take ℕ (evens) ∪ ℕ (odds) = ℕ. More precisely: ℕ × {0} ∪ ℕ × {1} ~ ℕ via the bijection (n,0)↦2n and (n,1)↦2n+1. So two disjoint countable sets together are still countable!

Cardinal Multiplication

Definition — Cardinal Multiplication

|A| × |B| = |A × B| — the cardinality of the Cartesian product.

Rules for Cardinal Multiplication
  • m × n = ordinary product for finite cardinals.
  • ℵ₀ × n = ℵ₀ for any finite n > 0.
  • ℵ₀ × ℵ₀ = ℵ₀ — ℕ × ℕ is countable! (Cantor's diagonal grid)
  • 𝔠 × 𝔠 = 𝔠 — the plane ℝ² has same cardinality as ℝ.
  • 0 × κ = 0 for any cardinal κ.
  • For infinite κ: κ × κ = κ (assuming Choice).
Example — ℵ₀ × ℵ₀ = ℵ₀

ℕ × ℕ = {(0,0),(0,1),(1,0),(0,2),(1,1),(2,0),(0,3),…}
Enumerate by diagonals: pairs (m,n) with m+n=0 first, then m+n=1, etc.
This lists all pairs — so ℕ × ℕ is countable, |ℕ × ℕ| = ℵ₀.

Cardinal Exponentiation

Definition — Cardinal Exponentiation

|A|^{|B|} = |A^B| where A^B is the set of all functions from B to A.

Key Results
  • 2^{ℵ₀} = 𝔠 = |ℝ| — functions from ℕ to {0,1} correspond to real numbers (binary expansions).
  • n^{ℵ₀} = 𝔠 for any finite n ≥ 2.
  • ℵ₀^{ℵ₀} = 𝔠
  • 2^κ > κ for any cardinal κ (Cantor's theorem).

⚠️ Key Intuition Break

In ordinary arithmetic: n + n > n and n × n > n. For infinite cardinals: ℵ₀ + ℵ₀ = ℵ₀ and ℵ₀ × ℵ₀ = ℵ₀. Only exponentiation produces a strictly larger cardinal: 2^{ℵ₀} > ℵ₀.

📝 Proof Templates

Proof 1 — ℕ ~ ℤ
Claim: The integers are countably infinite.
Define f: ℕ → ℤ by: f(n) = n/2 if n is even f(n) = -(n+1)/2 if n is odd Injective: If f(m)=f(n), case analysis shows m=n. ✓ Surjective: For z ≥ 0: z = f(2z) since 2z is even. ✓ For z < 0: z = f(-2z-1) since -2z-1 is odd. ✓ Therefore f is a bijection, so ℕ ~ ℤ. □
Proof 2 — Equinumerosity is Transitive
Claim: If A~B and B~C then A~C.
Proof: A~B means ∃ bijection f: A→B. B~C means ∃ bijection g: B→C. Define h: A→C by h = g∘f (composition). h is injective: h(a₁)=h(a₂) ⟹ g(f(a₁))=g(f(a₂)) ⟹ f(a₁)=f(a₂) [g injective] ⟹ a₁=a₂ [f injective]. ✓ h is surjective: Let c∈C. Since g surjective, ∃b∈B: g(b)=c. Since f surjective, ∃a∈A: f(a)=b. Then h(a)=g(f(a))=g(b)=c. ✓ Therefore h is a bijection, so A~C. □

📊 Self-Assessment — 10 Questions

1. Two sets A and B are equinumerous if:
A A and B have the same elements
B |A| and |B| are both infinite
C There exists a bijection f: A → B
D A is a subset of B
✅ C is correct. Equinumerosity means a bijection (perfect one-to-one correspondence) exists. Same elements would make them equal sets, not just equinumerous.
2. Which of the following sets has cardinality ℵ₀?
A {1, 2, 3, …, 1000}
B The set of all real numbers ℝ
C 𝒫(ℕ) (power set of ℕ)
D The set of all integers ℤ
✅ D is correct. ℤ ~ ℕ via the interleaving bijection, so |ℤ| = ℵ₀. ℝ and 𝒫(ℕ) have cardinality 𝔠 = 2^{ℵ₀} > ℵ₀. {1,...,1000} is finite.
3. The function f: ℕ → E (even numbers) defined by f(n) = 2n proves:
A The even numbers are uncountable
B The even numbers are equinumerous to ℕ
C There are fewer even numbers than natural numbers
D ℕ is a subset of the even numbers
✅ B is correct. f(n)=2n is a bijection ℕ→E (injective and surjective), so |E| = |ℕ| = ℵ₀. The even numbers, despite being a proper subset of ℕ, are equinumerous to ℕ.
4. Cantor's diagonal argument proves that:
A ℚ is uncountable
B ℕ and ℤ are equinumerous
C ℝ is uncountable (no bijection ℕ → ℝ exists)
D Every infinite set is countable
✅ C is correct. Cantor's diagonal argument assumes a complete list of reals exists and constructs a real number NOT on the list — proving no such list can exist, i.e., ℝ is uncountable.
5. What is ℵ₀ + ℵ₀?
A 2 · ℵ₀ (a new, larger cardinal)
B ℵ₁
C ℵ₀
D 𝔠
✅ C is correct. ℵ₀ + ℵ₀ = ℵ₀. Two countably infinite disjoint sets together form a countably infinite set (interleave their elements). Cardinal addition behaves very differently from ordinary addition!
6. What is 2^{ℵ₀}?
A ℵ₀
B 𝔠 (the cardinality of ℝ)
C ℵ₁
D ℵ₀²
✅ B is correct. 2^{ℵ₀} = 𝔠 = |ℝ|. The cardinality of all functions from ℕ to {0,1} equals the cardinality of the reals (think binary expansions).
7. Equinumerosity is an equivalence relation because it is:
A Reflexive only
B Symmetric and transitive only
C Reflexive and symmetric only
D Reflexive, symmetric, and transitive
✅ D is correct. Reflexive: A~A (identity). Symmetric: A~B ⟹ B~A (inverse bijection). Transitive: A~B and B~C ⟹ A~C (compose bijections). All three hold!
8. Which correctly describes the ordering of cardinals?
A ℵ₀ > 𝔠
B ℵ₀ = 𝔠
C ℵ₀ < 𝔠 < 2^𝔠
D 𝔠 < ℵ₀
✅ C is correct. ℵ₀ < 𝔠 (Cantor's diagonal argument). 𝔠 < 2^𝔠 (Cantor's theorem applied to 𝔠). There's an infinite tower of strictly increasing cardinals!
9. A set A is countable if and only if:
A A is finite
B A is finite or equinumerous to ℕ
C A is infinite
D A is equinumerous to ℝ
✅ B is correct. "Countable" includes both finite sets AND countably infinite sets (those with |A| = ℵ₀). Uncountable means infinite AND NOT equinumerous to ℕ.
10. Cantor's theorem states that for any set A:
A |A| = |𝒫(A)|
B |A| > |𝒫(A)|
C |A| < |𝒫(A)|
D |𝒫(A)| = ℵ₀ always
✅ C is correct. |A| < |𝒫(A)| for every set A. This means there is no largest cardinal — you can always get a bigger set by taking the power set!
