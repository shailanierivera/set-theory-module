Relations and Functions Between Sets

Big Picture: Before we define functions, we need a more general idea — a relation. Think of a relation as a rule that says "this element is connected to that element." A function is just a relation with a very specific rule: every input connects to exactly one output.

Step 1 — Building Blocks: Ordered Pairs

Definition — Ordered Pair

An ordered pair (a, b) is two objects listed in order. Order matters:
(a, b) = (c, d) if and only if a = c and b = d.
In particular, (1, 2) ≠ (2, 1) — these are different pairs.

Everyday Example

GPS coordinates: (latitude, longitude). The pair (14.5995° N, 120.9842° E) points to Manila. Swapping gives a different location entirely — order matters!

Step 2 — The Cartesian Product

Definition — Cartesian Product

For sets A and B, the Cartesian product is all possible ordered pairs:
A × B = { (a, b) | a ∈ A and b ∈ B }

Key Properties
  • |A × B| = |A| · |B| — the number of pairs equals the product of the set sizes.
  • A × B ≠ B × A in general (unless A = B).
  • A × ∅ = ∅ — no pairs are possible with an empty set.
  • A × (B ∪ C) = (A × B) ∪ (A × C) — distributes over union.
Example

Let A = {1, 2} and B = {a, b, c}.
A × B = { (1,a), (1,b), (1,c), (2,a), (2,b), (2,c) }
|A × B| = 2 × 3 = 6 pairs.
Notice: B × A = { (a,1),(a,2),(b,1),(b,2),(c,1),(c,2) } — totally different pairs!

Step 3 — Relations

Definition — Binary Relation

A binary relation R from set A to set B is any subset:
R ⊆ A × B
We write a R b (or (a,b) ∈ R) to say "a is related to b under R."
A relation on A means both sets are the same: R ⊆ A × A.

Example 1 — "Is a sibling of"

Let A = {Ana, Ben, Clara, Diego} where Ana & Ben are siblings, and Clara & Diego are siblings.
The "is a sibling of" relation is: R = {(Ana,Ben),(Ben,Ana),(Clara,Diego),(Diego,Clara)}
Note: this relation is on A (same set).

Example 2 — "Divides" on ℕ

Define a R b iff a | b (a divides b) on {1,2,3,4,6}.
R = {(1,1),(1,2),(1,3),(1,4),(1,6),(2,2),(2,4),(2,6),(3,3),(3,6),(4,4),(6,6)}

Example 3 — "Less than" on {1,2,3}

R = {(1,2),(1,3),(2,3)} — all pairs where the first is strictly less than the second.

Step 4 — Functions (A Special Relation)

Definition — Function

A function f: A → B is a relation R ⊆ A × B satisfying two conditions:
1. Total: Every element of A appears as a first component in some pair.
2. Single-valued: Each element of A appears in at most one pair.
Together: each a ∈ A maps to exactly one b ∈ B, written f(a) = b.

Two-Part Test: Is R a function?
  • Totality: Does every domain element have at least one mapping?
  • Single-valuedness: Does every domain element have at most one mapping?

Fail either ⟹ not a function.

Example — Deciding Function vs. Not

Let A = {1,2,3}, B = {p,q,r}. Which of these is a function?

  1. R₁ = {(1,p),(2,q),(3,r)} — ✅ Function. Each of 1, 2, 3 maps to exactly one target.
  2. R₂ = {(1,p),(1,q),(2,r),(3,p)} — ❌ Not a function. Element 1 maps to both p and q.
  3. R₃ = {(1,p),(2,p),(3,p)} — ✅ Function. Multiple elements can share the same output — that's fine!
  4. R₄ = {(1,p),(2,q)} — ❌ Not a function. Element 3 has no mapping at all.
💡 Analogy

A function is like an assignment of teachers to classes. Each class (domain element) must have exactly one assigned teacher. Multiple classes can share the same teacher (that's fine), but no class can have two teachers, and no class can be left without one.

Image and Inverse Image

Big Picture: A function doesn't just map individual elements — it can "push" entire subsets forward, and "pull" subsets backward. These are called the image and the inverse image (pre-image).

The Image (Forward Direction)

Definition — Image of a Subset

For f: A → B and a subset S ⊆ A:
f(S) = { f(x) | x ∈ S } = { b ∈ B | b = f(x) for some x ∈ S }
Read: "the set of all outputs of f when inputs come from S."

Example 1 — Finite Set

Let f: {1,2,3,4} → {a,b,c} defined by f(1)=a, f(2)=a, f(3)=b, f(4)=c.
f({1,2}) = {a} — both 1 and 2 map to a, so image is just {a}.
f({2,3,4}) = {a,b,c} — covers all outputs.
f({1,3}) = {a,b} — 1→a, 3→b.

Example 2 — f(x) = x² on ℝ

f([-3, 3]) = [0, 9] — squaring [-3,3] gives outputs in [0,9].
f({-2, -1, 0, 1, 2}) = {0, 1, 4} — outputs after squaring.
f([1, 2]) = [1, 4] — positive inputs squared.

The Inverse Image (Backward Direction)

Definition — Inverse Image (Pre-image)

For f: A → B and a subset T ⊆ B:
f⁻¹(T) = { x ∈ A | f(x) ∈ T }
Read: "all inputs whose output lands inside T."
Important: This notation does NOT mean f must be invertible — f⁻¹(T) is defined for ANY function.

Example 3 — Finite Function

Using the same f(1)=a, f(2)=a, f(3)=b, f(4)=c:
f⁻¹({a}) = {1, 2} — both 1 and 2 map into {a}.
f⁻¹({b,c}) = {3, 4} — 3 maps to b, 4 maps to c.
f⁻¹({a,b,c}) = {1,2,3,4} — everything maps somewhere in {a,b,c}.
f⁻¹(∅) = ∅ — nothing maps into the empty set.

Example 4 — f(x) = x² on ℝ

f⁻¹({4, 9}) = {-3, -2, 2, 3} — all x with x²=4 or x²=9.
f⁻¹([0, 4]) = [-2, 2] — all x with 0 ≤ x² ≤ 4.
f⁻¹((-∞, 0)) = ∅ — x² is never negative.

Key Properties

Properties of Image f(S)
  • f(S₁ ∪ S₂) = f(S₁) ∪ f(S₂) — union always distributes. ✅
  • f(S₁ ∩ S₂) ⊆ f(S₁) ∩ f(S₂) — only subset in general. ⚠️
  • Equality holds for intersection when f is injective.
  • S₁ ⊆ S₂ ⟹ f(S₁) ⊆ f(S₂) — monotone.
  • f(A) is the range of f.
Properties of Inverse Image f⁻¹(T)
  • f⁻¹(T₁ ∪ T₂) = f⁻¹(T₁) ∪ f⁻¹(T₂) — union distributes exactly. ✅
  • f⁻¹(T₁ ∩ T₂) = f⁻¹(T₁) ∩ f⁻¹(T₂) — intersection distributes exactly. ✅
  • f⁻¹(Bᶜ \ T) = A \ f⁻¹(T) — complements distribute exactly. ✅
  • Inverse image is better behaved than the forward image!
Fundamental Inclusions

S ⊆ f⁻¹(f(S)) for any S ⊆ A — equality iff f is injective.
f(f⁻¹(T)) ⊆ T for any T ⊆ B — equality iff f is surjective.

⚠️ Counterexample — Intersection Fails

Let f(1)=a, f(2)=a. Let S₁={1}, S₂={2}.
S₁ ∩ S₂ = ∅, so f(S₁ ∩ S₂) = f(∅) = ∅.
But f(S₁) ∩ f(S₂) = {a} ∩ {a} = {a} ≠ ∅.
So f(S₁ ∩ S₂) ⊊ f(S₁) ∩ f(S₂) — strict subset!

Domain, Codomain, and Range

Big Picture: When we write f: A → B, three sets are at play — the inputs (A), the declared target (B), and the actual outputs (range). These are NOT always the same, and the distinction matters a lot.

Three Key Sets

Domain

The domain of f: A → B is the set A — every element here must have exactly one output.
Sometimes written dom(f).

Codomain

The codomain is the set B declared in f: A → B. It is the intended target — outputs must land here, but not every element of B needs to be hit.

Range (Image of the Domain)

The range (or image) is the set of all actual outputs:
range(f) = f(A) = { f(a) | a ∈ A }
Always: range(f) ⊆ codomain. Equal only when f is surjective.

Example 1 — Finite Function

Let f: {1,2,3} → {a,b,c,d} with f(1)=a, f(2)=a, f(3)=b.
Domain = {1,2,3}
Codomain = {a,b,c,d}
Range = {a,b} — only a and b are ever hit. c and d are "unused."

Example 2 — Real Function f(x) = x²

Defined as f: ℝ → ℝ (codomain declared as all reals).
Domain = ℝ
Codomain = ℝ
Range = [0, ∞) — x² is never negative, so only non-negative reals are outputs.
Range ≠ Codomain — f is NOT surjective.

Example 3 — f(x) = 2x on ℝ

Domain = ℝ, Codomain = ℝ, Range = ℝ (every real y = 2x for x = y/2).
Here Range = Codomain — f is surjective onto ℝ.

Example 4 — Domain of a Relation

For a relation R = {(1,a),(3,b)} ⊆ {1,2,3} × {a,b,c}:
dom(R) = {1,3} — only elements that appear as first components.
range(R) = {a,b} — only elements that appear as second components.
Note: element 2 is in the full set A but not in dom(R). This relation is not a function!

Important Note on Codomain vs. Range

Why the Distinction Matters
  • Two functions can have the same rule but different codomains, making them different functions mathematically.
  • Example: g: ℝ → [0,∞) with g(x) = x² is surjective; f: ℝ → ℝ with the same rule is NOT surjective.
  • The codomain is part of the function's identity.

Equivalence Relations, Order Relations & Partitions

Big Picture: Not all relations are functions. Some relations on a single set have special structure. The most important: equivalence relations (which group similar things) and order relations (which rank things).

Equivalence Relations

Definition — Equivalence Relation

A relation R on set A is an equivalence relation (written ~) if it is:
1. Reflexive: ∀a: a ~ a
2. Symmetric: a ~ b ⟹ b ~ a
3. Transitive: a ~ b and b ~ c ⟹ a ~ c

Example 1 — "Same remainder mod 3" on ℤ

Define a ~ b iff 3 | (a - b).
Reflexive: 3 | (a-a) = 0 ✅
Symmetric: if 3|(a-b) then 3|(b-a) ✅
Transitive: if 3|(a-b) and 3|(b-c) then 3|(a-c) ✅
So this IS an equivalence relation.

Example 2 — "Same first letter" on words

Two English words are equivalent if they start with the same letter.
Reflexive: "apple" starts with same letter as "apple" ✅
Symmetric: if "apple"~"ant" then "ant"~"apple" ✅
Transitive: "apple"~"ant" and "ant"~"anchor" ⟹ "apple"~"anchor" ✅
Valid equivalence relation!

❌ Not an Equivalence — "Less than" on ℝ

The relation "<" fails reflexivity (a ≮ a) and symmetry (a<b does NOT imply b<a). Not an equivalence relation.

Equivalence Classes and Partitions

Definition — Equivalence Class

For a ∈ A, the equivalence class of a is:
[a] = { x ∈ A | x ~ a } — all elements equivalent to a.
Also written ā or [a]_R.

Definition — Partition

A partition of A is a collection of subsets {A₁, A₂, …} such that:
1. Each Aᵢ ≠ ∅ (non-empty)
2. Aᵢ ∩ Aⱼ = ∅ for i ≠ j (pairwise disjoint)
3. A₁ ∪ A₂ ∪ … = A (covers everything)

Fundamental Theorem

Every equivalence relation on A creates a partition of A (the equivalence classes), and every partition of A defines an equivalence relation. These two concepts are in perfect correspondence.

Example — Mod 3 Classes

Equivalence classes of ℤ under mod 3:
[0] = {…,-6,-3,0,3,6,…}
[1] = {…,-5,-2,1,4,7,…}
[2] = {…,-4,-1,2,5,8,…}
These three classes partition ℤ: disjoint and their union is all of ℤ.

Order Relations

Partial Order (≤)

A relation R on A is a partial order if it is reflexive, antisymmetric, and transitive.
Example: divisibility (a|b) on ℕ, subset inclusion (A⊆B) on sets.

Strict Order (<)

A relation R is a strict order if it is irreflexive, asymmetric, and transitive.
Example: the usual "<" on ℝ, proper subset "⊊".

Total (Linear) Order

A partial order where every two elements are comparable: for all a,b either aRb or bRa.
Example: ≤ on ℝ is a total order. Divisibility on ℕ is NOT total (e.g., 2 and 3 are not comparable).

Inclusion Maps, Restriction Maps & Characteristic Functions

Big Picture: These are three useful "built-in" maps that come naturally from a set and its subsets. They let us zoom in on parts of a function, embed subsets into larger sets, and convert set membership into a number (0 or 1).

1. The Inclusion Map

Definition — Inclusion Map

For sets A ⊆ B, the inclusion map (also called the embedding) is:
ι: A ↪ B, defined by ι(a) = a for all a ∈ A
The hooked arrow ↪ signals it is always injective.

Properties of the Inclusion Map
  • Always injective — distinct elements of A stay distinct in B.
  • Surjective iff A = B.
  • Bijective iff A = B.
  • Composition: if A ⊆ B ⊆ C, then ι_{B,C} ∘ ι_{A,B} = ι_{A,C}.
Example 1 — Numbers

The natural numbers embed in the integers: ι: ℕ ↪ ℤ by ι(n) = n.
The integers embed in the rationals: ι: ℤ ↪ ℚ by ι(z) = z.
Each embedding is injective but not surjective (e.g., -1 ∉ ℕ; ½ ∉ ℤ).

Example 2 — Finite Set

Let A = {1, 3, 5} ⊆ B = {1, 2, 3, 4, 5}.
ι(1) = 1, ι(3) = 3, ι(5) = 5.
As a set of pairs: ι = {(1,1), (3,3), (5,5)}.

2. The Restriction Map

Definition — Restriction

For f: A → B and S ⊆ A, the restriction of f to S is:
f|_S: S → B, defined by f|_S(x) = f(x) for all x ∈ S
Same rule as f, but only applies to inputs in S.

Key Properties
  • If f is injective, then f|_S is injective.
  • f|_S may or may not be surjective even if f is.
  • f|_A = f (restricting to the full domain changes nothing).
  • f|_S = f ∘ ι where ι: S ↪ A is the inclusion map.
Example 3 — Restriction

Let f: ℝ → ℝ by f(x) = x². Restrict to S = [0, ∞).
f|_{[0,∞)}: [0,∞) → ℝ by f|_{[0,∞)}(x) = x².
On the full domain ℝ, f is NOT injective (f(-2)=f(2)=4). But f|_{[0,∞)} IS injective! Restriction can "fix" injectivity.

Example 4 — Finite

Let f: {1,2,3,4,5} → {a,b,c} with f(1)=a, f(2)=b, f(3)=a, f(4)=c, f(5)=b.
S = {1,3,4}. Then f|_S = {(1,a),(3,a),(4,c)}.

3. Characteristic (Indicator) Functions

Definition — Characteristic Function

For a set A and subset S ⊆ A, the characteristic function (or indicator function) is:
χ_S: A → {0, 1}

χ_S(x) = 1 if x ∈ S χ_S(x) = 0 if x ∉ S
Properties — Connecting Sets and Logic
  • χ_{A∩B}(x) = χ_A(x) · χ_B(x) — intersection = multiplication
  • χ_{A∪B}(x) = χ_A(x) + χ_B(x) − χ_A(x)·χ_B(x) — inclusion-exclusion
  • χ_{Aᶜ}(x) = 1 − χ_A(x) — complement = 1 minus
  • χ_∅(x) = 0 and χ_A(x) = 1 for the full set A
Example 5 — Characteristic Function

Let A = {1,2,3,4,5,6}, S = {2,4,6} (even numbers).
χ_S(1)=0, χ_S(2)=1, χ_S(3)=0, χ_S(4)=1, χ_S(5)=0, χ_S(6)=1
This is just a 0-or-1 flag for membership! The sum χ_S(1)+…+χ_S(6) = 3 equals |S|.

💡 Why Does This Matter?

Characteristic functions are the bridge between sets and numbers. In probability theory, χ_S is an indicator random variable. In computer science, it's a Boolean flag. In integration, ∫χ_S dμ = μ(S) (measure of S).

What is ι: A↪B?
Inclusion map: ι(a)=a. Sends A into B as itself. Always injective.
What is f|_S?
Restriction to S⊆A: same rule f, but only inputs from S are allowed.
χ_S(x) = ?
1 if x∈S, 0 if x∉S. Encodes set membership as 0 or 1.
χ_{A∩B} in terms of products?
χ_A(x) · χ_B(x) — both must be 1 for intersection to be 1.
Can restriction fix injectivity?
YES — restricting a non-injective function to a smaller domain can make it injective.

Injective, Surjective, Bijective & Composition

Big Picture: Functions can be classified by how they behave — do different inputs always produce different outputs? Is every possible output hit? Both? These three properties — injective, surjective, bijective — are fundamental to understanding function structure.

Injective (One-to-one)

Definition — Injective

f: A → B is injective (one-to-one) if:
f(a₁) = f(a₂) ⟹ a₁ = a₂
Equivalently (contrapositive): a₁ ≠ a₂ ⟹ f(a₁) ≠ f(a₂)
Plain: no two different inputs share the same output.

Example 1 — Injective Functions
  • f: ℝ → ℝ, f(x) = 2x + 3 — injective. If 2x+3 = 2y+3 then x=y. ✅
  • f: {1,2,3} → {a,b,c,d} with f(1)=a, f(2)=b, f(3)=c — injective. ✅
  • f: ℝ → ℝ, f(x) = eˣ — injective (exponential is strictly increasing). ✅
❌ Not Injective

f(x) = x² on ℝ: f(-2) = f(2) = 4. Two different inputs, same output!

Surjective (Onto)

Definition — Surjective

f: A → B is surjective (onto) if:
∀ b ∈ B, ∃ a ∈ A: f(a) = b
Plain: every element of B is hit by at least one element of A. Range = Codomain.

Example 2 — Surjective Functions
  • f: ℝ → ℝ, f(x) = 2x + 3 — surjective. For any y, x = (y-3)/2 works. ✅
  • f: {1,2,3} → {a,b} with f(1)=a, f(2)=b, f(3)=a — surjective (a and b both hit). ✅
  • f(x) = x³ on ℝ — surjective. Every real y is reached by x = ∛y. ✅
❌ Not Surjective

f: ℝ → ℝ, f(x) = x² — the output -1 is never reached. Range = [0,∞) ≠ ℝ.

Bijective (Perfect Correspondence)

Definition — Bijective

f: A → B is bijective if it is both injective AND surjective.
Plain: every element of B is hit by exactly one element of A.
A bijection has an inverse function f⁻¹: B → A.

Example 3 — Bijective
  • f: ℝ → ℝ, f(x) = 2x + 3 — bijective. ✅
  • f: {1,2,3} → {a,b,c} with f(1)=a, f(2)=b, f(3)=c — bijective. ✅
  • f(x) = x³ on ℝ — bijective. ✅
  • The identity map id_A: A → A, id(a) = a — always bijective. ✅

Composition of Functions

Definition — Composition

For f: A → B and g: B → C, the composition is:
(g ∘ f): A → C, (g ∘ f)(x) = g(f(x))
Read "g after f" or "g composed with f." Apply f FIRST, then g.

Example 4 — Composition

Let f(x) = 2x + 1 and g(x) = x².
(g ∘ f)(x) = g(f(x)) = g(2x+1) = (2x+1)²
(f ∘ g)(x) = f(g(x)) = f(x²) = 2x²+1
Note: g∘f ≠ f∘g in general — composition is NOT commutative!

Theorems on Composition
  • f and g injective ⟹ g∘f injective
  • f and g surjective ⟹ g∘f surjective
  • f and g bijective ⟹ g∘f bijective (and (g∘f)⁻¹ = f⁻¹∘g⁻¹)
  • g∘f injective ⟹ f is injective (but g need not be)
  • g∘f surjective ⟹ g is surjective (but f need not be)
  • Composition is associative: h∘(g∘f) = (h∘g)∘f
Example 5 — Why g∘f Injective Doesn't Force g Injective

Let A={1}, B={1,2}, C={a}. Define f(1)=1, g(1)=a, g(2)=a.
g∘f: A→C is injective (only one element in A!). But g is NOT injective (g(1)=g(2)=a).

📝 Proof Templates

Proof 1 — Image of Union
Claim: f(S₁ ∪ S₂) = f(S₁) ∪ f(S₂)
Proof (⊆): Let y ∈ f(S₁ ∪ S₂). Then ∃x ∈ S₁ ∪ S₂ with f(x) = y. Case 1: x ∈ S₁ ⟹ y = f(x) ∈ f(S₁) ⊆ f(S₁) ∪ f(S₂). Case 2: x ∈ S₂ ⟹ y = f(x) ∈ f(S₂) ⊆ f(S₁) ∪ f(S₂). Either way, y ∈ f(S₁) ∪ f(S₂). ✓ Proof (⊇): Let y ∈ f(S₁) ∪ f(S₂). Case 1: y ∈ f(S₁) ⟹ ∃x ∈ S₁ ⊆ S₁∪S₂ with f(x)=y ⟹ y ∈ f(S₁∪S₂). Case 2: y ∈ f(S₂) ⟹ ∃x ∈ S₂ ⊆ S₁∪S₂ with f(x)=y ⟹ y ∈ f(S₁∪S₂). Either way, y ∈ f(S₁∪S₂). ✓ □
Proof 2 — g∘f injective ⟹ f injective
Claim: If g∘f is injective, then f is injective.
Proof (direct): Assume g∘f: A → C is injective. Let a₁, a₂ ∈ A with f(a₁) = f(a₂). [want to show a₁ = a₂] Apply g to both sides: g(f(a₁)) = g(f(a₂)) (g∘f)(a₁) = (g∘f)(a₂) Since g∘f is injective: a₁ = a₂. Therefore f is injective. □

5Es · Evaluate
Knowledge Check

12 questions covering all six topics. Answers are checked immediately. Read every explanation — even for correct answers!

📊 Self-Assessment — 12 Questions

1. Which of the following is a valid function from A = {1,2,3} to B = {a,b,c}?
A {(1,a),(1,b),(2,c),(3,a)}
B {(1,b),(2,a),(3,c)}
C {(1,a),(2,b)}
D {(2,a),(3,b),(3,c)}
✅ B is correct. Each of 1, 2, 3 maps to exactly one element. A: element 1 maps to both a and b (not single-valued). C: element 3 has no mapping (not total). D: element 3 maps to two outputs.
2. For f: ℝ→ℝ defined by f(x) = x², the inverse image f⁻¹({1, 4}) equals:
A {1, 2}
B {1, 4}
C {−2, −1, 1, 2}
D {−1, 1}
✅ C is correct. f⁻¹({1,4}) = {x | x²=1 or x²=4} = {-1,1,-2,2}. The negative values are included because (-1)²=1 and (-2)²=4.
3. Let f(1)=a, f(2)=a, f(3)=b with domain {1,2,3} and codomain {a,b,c}. What is the range of f?
A {a, b}
B {a, b, c}
C {1, 2, 3}
D {a}
✅ A is correct. Range = actual outputs = {f(1),f(2),f(3)} = {a,a,b} = {a,b}. Element c is in the codomain but never actually output.
4. A relation R on set A is an equivalence relation if and only if it is:
A Reflexive and Transitive
B Symmetric and Antisymmetric
C Reflexive, Symmetric, and Transitive
D Reflexive, Antisymmetric, and Transitive
✅ C is correct. All three properties are needed. Option D describes a partial order, not equivalence.
5. The function f: ℝ→ℝ, f(x) = x³ is:
A Injective only
B Surjective only
C Neither
D Bijective
✅ D is correct. f(x)=x³ is injective (x³=y³ implies x=y since cube root is unique) and surjective (every real y = (∛y)³ is achieved). So it's bijective.
6. The characteristic function χ_S on A={1,2,3,4,5} where S={1,3,5} gives χ_S(4) = ?
A 1
B 0
C 4
D Undefined
✅ B is correct. 4 ∉ S = {1,3,5}, so χ_S(4) = 0. The characteristic function outputs 1 only for elements IN S.
7. If g∘f is surjective, which must be true?
A f is surjective
B f is injective
C g is surjective
D g is injective
✅ C is correct. If g∘f hits every element of C, then g alone must hit every element of C — so g is surjective. But f need not be (it might not fill all of B).
8. Which property distinguishes a partial order from an equivalence relation?
A Partial orders are reflexive; equivalence relations are not
B Partial orders are antisymmetric; equivalence relations are symmetric
C Partial orders are not transitive; equivalence relations are
D Equivalence relations require more than 3 properties
✅ B is correct. Both are reflexive and transitive. The key difference: equivalence uses symmetry (aRb⟹bRa), while partial order uses antisymmetry (aRb AND bRa ⟹ a=b).
9. Let f: A→B and S ⊆ A. Which of the following is always true?
A f(f⁻¹(f(S))) = f(S)
B f⁻¹(f(S)) = S
C f(S) = B
D S ⊆ f⁻¹(f(S))
✅ D is correct. S ⊆ f⁻¹(f(S)) is always true — every element x∈S satisfies f(x)∈f(S), so x∈f⁻¹(f(S)). Option B requires injectivity; C requires surjectivity.
10. The inclusion map ι: A↪B (where A ⊊ B, A ≠ B) is:
A Bijective
B Surjective but not injective
C Injective but not surjective
D Neither injective nor surjective
✅ C is correct. ι is injective (different elements stay different), but NOT surjective because A ⊊ B means there are elements in B not in A that ι cannot reach.
11. Which relation on ℤ is an equivalence relation?
A a R b iff a < b
B a R b iff a divides b
C a R b iff a ≡ b (mod 4)
D a R b iff a + b = 0
✅ C is correct. Congruence mod 4 is reflexive (a≡a), symmetric (a≡b⟹b≡a), and transitive. Option A fails reflexivity and symmetry. B fails symmetry. D fails reflexivity (a+a=0 only for a=0).
12. Let f(x) = 2x and g(x) = x+5. Then (f∘g)(3) = ?
A 11
B 16
C 13
D 6
✅ B is correct. (f∘g)(3) = f(g(3)) = f(3+5) = f(8) = 2×8 = 16. Remember: apply g FIRST (inside), then f.
