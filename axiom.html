The Axiom of Choice

Big Picture: The Axiom of Choice (AC) says you can always simultaneously pick one element from each set in a collection of non-empty sets — even infinitely many of them. This sounds obvious, but it has surprising and even paradoxical consequences.

The Axiom

Axiom of Choice (AC)

For any collection 𝒞 of non-empty sets, there exists a choice function f such that for each set S ∈ 𝒞, we have f(S) ∈ S.

In plain language: given any collection of non-empty sets (even infinitely many), you can always pick exactly one element from each set.

Why Is It Needed?

For finite collections, no axiom is needed — we can just make each choice one by one. For countably infinite collections, we can describe an explicit rule. For uncountably infinite collections with no clear rule, the Axiom of Choice steps in and asserts a choice function exists — even if we can never explicitly describe it.
Example 1 — Shoes vs. Socks (Bertrand Russell)

Infinitely many pairs of shoes: pick the left shoe from each pair. No AC needed — we have an explicit rule.
Infinitely many pairs of identical socks: socks have no left/right distinction. There is NO explicit rule for picking one from each pair. The Axiom of Choice asserts we can still make the choices — but we can't describe how.

Example 2 — Simple Use of AC

Let 𝒞 = { {1,2}, {3,4,5}, {6}, {7,8} }. A choice function might be f({1,2})=1, f({3,4,5})=4, f({6})=6, f({7,8})=7. For finite collections, AC is trivially true. It becomes essential for infinite collections where no explicit rule exists.

Example 3 — Choosing from Equivalence Classes

Define the equivalence relation on ℝ: x ~ y iff x - y ∈ ℚ. This partitions ℝ into uncountably many equivalence classes. AC says we can pick one representative from each class — this is the Vitali set construction (which produces a non-measurable set!).

Equivalent Formulations

AC is Equivalent To Each of These
  • Zorn's Lemma: Every partially ordered set where every chain has an upper bound has a maximal element.
  • Well-Ordering Theorem: Every set can be well-ordered (every non-empty subset has a least element).
  • Tychonoff's Theorem: Any product of compact topological spaces is compact.
  • Vector Space Basis: Every vector space has a basis.
  • Cardinal Comparability: For any two cardinals κ and λ, either κ ≤ λ or λ ≤ κ.

Consequences of AC

Results That Require AC
  • Every infinite set contains a countably infinite subset.
  • A countable union of countable sets is countable.
  • Every surjective function has a right inverse.
  • Every field has an algebraic closure.
  • The Schröder–Bernstein theorem (in some formulations).
  • ℵ₀ + ℵ₀ = ℵ₀ (for arbitrary infinite cardinals).
⚠️ The Banach–Tarski Paradox

Using AC, it can be proven that a solid sphere in ℝ³ can be decomposed into finitely many pieces and reassembled into TWO spheres of the same size. This is not a physical claim — it relies on non-measurable sets that only exist because of AC. It shows AC can lead to highly counter-intuitive results.

Is AC True?

Independence of AC from ZF

Gödel (1938) showed AC is consistent with ZF set theory — assuming ZF is consistent, adding AC creates no contradictions.
Cohen (1963) showed AC is independent of ZF — it cannot be proved from ZF alone.
Together: AC can be freely added to ZF. The system ZF + AC = ZFC is the standard foundation of modern mathematics.

The Continuum Hypothesis

Big Picture: We know ℵ₀ < 𝔠. The Continuum Hypothesis asks: is there a set with cardinality strictly between these two? This is one of the most famous unsolved — and unsolvable — questions in mathematics.

Background — The Cardinal Ladder

From the previous module, we know there are different sizes of infinity. The natural numbers have cardinality ℵ₀, and the real numbers have cardinality 𝔠 = 2^{ℵ₀}. Cantor proved ℵ₀ < 𝔠 — these are genuinely different infinities.

The Continuum Hypothesis (CH)

There is no set whose cardinality is strictly between that of the natural numbers and the real numbers.

Formally: 2^{ℵ₀} = ℵ₁

Here ℵ₁ is the next cardinal after ℵ₀. CH claims the continuum 𝔠 IS that next cardinal — there is nothing in between.

Generalized Continuum Hypothesis (GCH)

For every infinite cardinal κ: 2^κ = κ⁺ (the next cardinal after κ).
In other words, there is no cardinal strictly between κ and 2^κ for any infinite cardinal.

What Would CH Mean?

If CH is True

The cardinal hierarchy looks like: ℵ₀ < ℵ₁ = 𝔠 < ℵ₂ < …
Every infinite subset of ℝ is either countable (size ℵ₀) or has the full cardinality of ℝ (size 𝔠). There is no "intermediate" infinity between the naturals and reals.

If CH is False

The cardinal hierarchy looks like: ℵ₀ < ℵ₁ < 𝔠 (or even more cardinals in between).
There exists a set S ⊆ ℝ that is uncountable but "smaller" than ℝ. We cannot construct such a set explicitly — its existence would be asserted non-constructively.

The Independence Result

Gödel–Cohen Independence Theorem

Gödel (1938): CH is consistent with ZFC. If ZFC has no contradictions, then neither does ZFC + CH. (You can assume CH is true without breaking mathematics.)

Cohen (1963): ¬CH is also consistent with ZFC. If ZFC has no contradictions, then neither does ZFC + ¬CH. (You can also assume CH is false without breaking mathematics.)

Conclusion: CH is independent of ZFC — it can neither be proved nor disproved from the standard axioms of set theory.

⚠️ What Does "Independent" Mean?

Independence doesn't mean "we don't know yet." It means the question is fundamentally undecidable within ZFC. There are consistent mathematical universes where CH is true, and consistent universes where CH is false. Both are equally valid — the standard axioms do not determine which universe we are in.

Historical Context

Timeline
  • 1878 — Cantor first formulated CH as a conjecture in his work on cardinality.
  • 1900 — Hilbert listed CH as the first of his famous 23 problems.
  • 1938 — Gödel proved CH is consistent with ZFC (cannot be disproved).
  • 1963 — Cohen proved ¬CH is consistent with ZFC (cannot be proved). Won the Fields Medal.
  • Today — Mathematicians explore new axioms (like large cardinal axioms) that may give a definitive answer.

Constructing ℤ, ℚ, and ℝ from Sets

Big Picture: All of mathematics can be built from sets. Starting with the natural numbers ℕ (assumed as a foundation), we construct ℤ, ℚ, and ℝ using purely set-theoretic tools — equivalence classes, ordered pairs, and Dedekind cuts. This shows that numbers themselves are just sets in disguise.

Step 1 — Constructing ℤ (The Integers)

Problem: ℕ has no negative numbers. We can't solve n + x = m when m < n. Solution: Think of an integer as a "difference" of two natural numbers — encode it as a pair (a, b) meaning "a − b."
Construction of ℤ

Step 1: Form the set ℕ × ℕ = {(a,b) | a,b ∈ ℕ}. Think of (a,b) as "a − b."

Step 2: Define an equivalence relation: (a,b) ~ (c,d) iff a + d = b + c (same "difference").

Step 3: The integers are the equivalence classes: ℤ = (ℕ × ℕ) / ~

Examples — Integer Construction
  • [(3,0)] = {(3,0),(4,1),(5,2),…} represents the integer +3
  • [(0,2)] = {(0,2),(1,3),(2,4),…} represents the integer −2
  • [(5,5)] = {(0,0),(1,1),(2,2),…} represents the integer 0
  • Why (3,1) ~ (5,3)? Check: 3+3 = 1+5 = 6 ✅ Both represent integer +2.
Arithmetic on ℤ (via representatives)
  • Addition: [(a,b)] + [(c,d)] = [(a+c, b+d)]
  • Negation: -[(a,b)] = [(b,a)] — swap the pair!
  • Multiplication: [(a,b)] × [(c,d)] = [(ac+bd, ad+bc)]
  • These are well-defined (same result regardless of representative chosen).

Step 2 — Constructing ℚ (The Rationals)

Problem: ℤ has no fractions. We can't solve n · x = m when n ∤ m. Solution: Encode a fraction as a pair (p, q) meaning "p/q" with q ≠ 0.
Construction of ℚ

Step 1: Form the set ℤ × ℤ* = {(p,q) | p ∈ ℤ, q ∈ ℤ, q ≠ 0}. Think of (p,q) as "p/q."

Step 2: Define equivalence: (p,q) ~ (r,s) iff p · s = q · r (cross-multiply).

Step 3: The rationals are the equivalence classes: ℚ = (ℤ × ℤ*) / ~

Examples — Rational Construction
  • [(1,2)] = {(1,2),(2,4),(3,6),(−1,−2),…} represents 1/2
  • [(3,4)] = {(3,4),(6,8),(−3,−4),…} represents 3/4
  • [(2,1)] = {(2,1),(4,2),(6,3),…} represents integer 2 — ℤ embeds into ℚ!
  • Why (1,2) ~ (2,4)? Check: 1×4 = 2×2 = 4 ✅ Both represent 1/2.
Arithmetic on ℚ
  • Addition: [(p,q)] + [(r,s)] = [(ps+qr, qs)] (common denominator)
  • Multiplication: [(p,q)] × [(r,s)] = [(pr, qs)]
  • Inverse: [(p,q)]⁻¹ = [(q,p)] for p ≠ 0
  • ℚ is a field — closed under +, −, ×, ÷ (except by 0).

Step 3 — Constructing ℝ (The Real Numbers)

Problem: ℚ has "gaps" — numbers like √2, π, e cannot be expressed as fractions. The rationals are not complete (Cauchy sequences can fail to converge in ℚ). Solution: Two main methods — Dedekind cuts or Cauchy sequences.
Method 1 — Dedekind Cuts

A Dedekind cut is a pair (A, B) that partitions ℚ into two non-empty sets where:
1. A ∪ B = ℚ, A ∩ B = ∅
2. Every element of A is less than every element of B
3. A has no maximum element

Each cut (A, B) represents a real number — the "boundary" between A and B.

Example — Dedekind Cut for √2

Define the cut for √2 as:
A = {q ∈ ℚ | q < 0 or q² < 2}
B = {q ∈ ℚ | q ≥ 0 and q² ≥ 2}
A contains all rationals "below √2" and B contains all rationals "at or above √2." The number √2 itself is irrational — it's represented by the gap between A and B. Every rational corresponds to a cut, but cuts can also capture irrationals!

Method 2 — Cauchy Sequences

A Cauchy sequence is a sequence (qₙ) in ℚ where elements get arbitrarily close together.
Define equivalence: (qₙ) ~ (rₙ) iff |qₙ − rₙ| → 0.
The real numbers are the equivalence classes of Cauchy sequences: ℝ = {Cauchy sequences in ℚ} / ~

Example — √2 via Cauchy Sequences

The sequence 1, 1.4, 1.41, 1.414, 1.4142, … is a Cauchy sequence in ℚ whose limit is √2 ∉ ℚ.
Its equivalence class IS the real number √2.
Similarly: π = [3, 3.1, 3.14, 3.141, 3.1415, …] and e = [2, 2.7, 2.71, 2.718, …]

Key Properties of ℝ
  • Completeness: Every Cauchy sequence of real numbers converges to a real number.
  • Ordered field: ℝ is a field with a total order compatible with arithmetic.
  • Archimedean property: For any x ∈ ℝ, there exists n ∈ ℕ with n > x.
  • Density of ℚ in ℝ: Between any two real numbers there is a rational number.
  • ℝ is the unique complete ordered field (up to isomorphism).

How is ℤ constructed?
ℤ = (ℕ×ℕ)/~ where (a,b)~(c,d) iff a+d=b+c. Pair (a,b) represents integer a−b.
How is ℚ constructed?
ℚ = (ℤ×ℤ*)/~ where (p,q)~(r,s) iff ps=qr. Pair (p,q) represents fraction p/q.
What is a Dedekind cut?
A partition (A,B) of ℚ where A has no max, every a∈A < every b∈B. Represents a real number.
What does (3,5) represent in ℤ?
The integer 3−5 = −2. Equivalence class [(3,5)] = {(0,2),(1,3),(2,4),(3,5),…}

The rational 2/6 = 1/3. Same class as (1,3),(3,9),(4,12),…

📝 Proof Templates

Proof 1 — Integer Addition is Well-Defined
Claim: If (a,b)~(a',b') and (c,d)~(c',d'), then (a+c,b+d)~(a'+c',b'+d').
Given: (a,b) ~ (a',b') means a + b' = a' + b ... (i) (c,d) ~ (c',d') means c + d' = c' + d ... (ii) Add (i) and (ii): (a + c) + (b' + d') = (a' + c') + (b + d) This is exactly the condition for: (a+c, b+d) ~ (a'+c', b'+d') ✓ Therefore addition is well-defined on ℤ. □
Proof 2 — ℚ is an Equivalence Relation
Claim: The relation (p,q)~(r,s) iff ps=qr is an equivalence relation on ℤ×ℤ*.
Reflexive: (p,q) ~ (p,q)? Need: p·q = q·p. ✓ (commutativity) Symmetric: If (p,q)~(r,s) then ps=qr. Need (r,s)~(p,q): rq = sp. From ps=qr: rq = ps = sp. ✓ Transitive: If (p,q)~(r,s) and (r,s)~(t,u): ps = qr and ru = st Multiply: ps·ru = qr·st psu·r = qst·r Since r ≠ 0: pu = qt So (p,q) ~ (t,u). ✓ □

📊 Self-Assessment — 12 Questions

1. The Axiom of Choice states that:
AEvery set can be well-ordered
BEvery infinite set is countable
CFor any collection of non-empty sets, a choice function exists
DEvery subset of ℝ is measurable
✅ C is correct. AC directly states a choice function exists for any collection of non-empty sets. A is the Well-Ordering Theorem (equivalent to AC, but not the statement of AC itself).
2. Which of the following is equivalent to the Axiom of Choice?
ACantor's diagonal argument
BZorn's Lemma
CThe Continuum Hypothesis
DSchröder–Bernstein Theorem
✅ B is correct. Zorn's Lemma, the Well-Ordering Theorem, and Tychonoff's Theorem are all equivalent to AC. CH is independent of ZFC; Cantor's diagonal argument is a proof technique.
3. The Continuum Hypothesis states that:
Aℝ is countable
BThere is no set with cardinality strictly between ℵ₀ and 𝔠
C2^{ℵ₀} = ℵ₂
DEvery infinite set has cardinality ℵ₀
✅ B is correct. CH says 2^{ℵ₀} = ℵ₁ — the next cardinal after ℵ₀. There's nothing between |ℕ| and |ℝ|. C is wrong (ℵ₂ would leave a gap). D is wrong — ℝ has a larger cardinality.
4. Who proved the independence of the Continuum Hypothesis from ZFC?
ACantor and Hilbert
BZermelo and Fraenkel
CGödel (1938) and Cohen (1963)
DDedekind and Cauchy
✅ C is correct. Gödel showed CH is consistent with ZFC (can't be disproved). Cohen showed ¬CH is also consistent (can't be proved). Together, CH is independent. Cohen won the Fields Medal for this.
5. In the construction of ℤ, the pair (5, 3) represents which integer?
A+2
B+8
C−2
D+5/3
✅ A is correct. The pair (a,b) in the construction of ℤ represents a − b = 5 − 3 = +2.
6. In the construction of ℚ, which pair is in the same equivalence class as (2, 6)?
A(2, 3)
B(4, 6)
C(2, 4)
D(1, 3)
✅ D is correct. (2,6) represents 2/6 = 1/3. Check (1,3): 1×6 = 2×3 = 6 ✓ — cross-multiplication confirms they're equivalent. (2,3) = 2/3, (4,6) = 4/6 = 2/3, (2,4) = 1/2.
7. A Dedekind cut (A, B) of ℚ represents a real number. Which condition must hold?
AA and B are both finite
BA has a maximum element
CA ∪ B = ℚ, A ∩ B = ∅, every a∈A < every b∈B, and A has no maximum
DBoth A and B are countable
✅ C is correct. These four conditions define a Dedekind cut. Crucially, A must have no maximum — otherwise the "boundary" would belong to A and wouldn't properly represent an irrational number.
8. In the formal construction, what does the equivalence relation for ℤ require?
A(a,b) ~ (c,d) iff a = c and b = d
B(a,b) ~ (c,d) iff a + d = b + c
C(a,b) ~ (c,d) iff a · d = b · c
D(a,b) ~ (c,d) iff a − b = c − d (using integer subtraction)
✅ B is correct. Since we're working in ℕ (no subtraction yet), we encode "a−b = c−d" as a+d = b+c — using only addition. D uses subtraction, which we haven't defined yet at this stage!
9. What property of ℝ does the Dedekind cut construction guarantee?
Aℝ is countable
Bℝ contains only rational numbers
Cℝ is complete — every Cauchy sequence converges in ℝ
Dℝ is finite
✅ C is correct. The whole point of constructing ℝ from ℚ is to fill the "gaps" — creating a complete ordered field where every Cauchy sequence converges.
10. Which correctly describes the relationship between number systems?
Aℝ ⊂ ℚ ⊂ ℤ ⊂ ℕ
Bℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ ⊂ ℂ
Cℤ ⊂ ℕ ⊂ ℚ ⊂ ℝ
Dℕ = ℤ = ℚ = ℝ
✅ B is correct. Each system extends the previous: ℕ (counting) ⊂ ℤ (adds negatives) ⊂ ℚ (adds fractions) ⊂ ℝ (adds irrationals) ⊂ ℂ (adds imaginary numbers).
11. The Axiom of Choice is independent of ZF. This means:
AAC is false
BAC is true but unproven
CAC contradicts ZF
DAC can neither be proved nor disproved from ZF alone
✅ D is correct. Independence means both ZF+AC and ZF+¬AC are consistent. You can choose to include or exclude AC without contradiction. Most mathematicians include it (ZFC).
12. The sequence 1, 1.7, 1.73, 1.732, … (converging to √3) represents which real number in the Cauchy sequence construction of ℝ?
AA rational number
BAn undefined value
CThe irrational number √3
DThe integer 2
✅ C is correct. In the Cauchy sequence construction of ℝ, a real number IS an equivalence class of Cauchy sequences. The sequence 1, 1.7, 1.73, … is a Cauchy sequence whose equivalence class defines √3 ∈ ℝ \ ℚ.
