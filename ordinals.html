Beyond Counting

Cardinal numbers tell us HOW MANY. But ordinals tell us something different — they capture ORDER and position. Let's discover why that distinction matters profoundly.

🤔 Cardinal vs. Ordinal Thinking

🏁
Race Positions

5 runners finish a race. The cardinal number 5 tells us how many ran. But "1st, 2nd, 3rd…" are ordinals — they encode the ORDER of finishing, not just the count.

🔢
What comes after all natural numbers?

After 0, 1, 2, 3, … there's a "first infinite ordinal" called ω. After ω comes ω+1, then ω+2, … then ω·2, then ω², then ω^ω, then ε₀…

🔁
Why can't ordinary induction reach ω?

Standard induction proves P(0) and P(n)→P(n+1). It reaches every finite n — but never ω. Transfinite induction fills that gap.

Ordinal Numbers

Big Picture: Cardinals measure SIZE. Ordinals measure ORDER. Two sets with the same cardinality can have different ordinal structure — depending on how their elements are arranged. Ordinals formalize the idea of "position in a sequence," even for infinite sequences.

Step 1 — Well-Ordered Sets

Definition — Well-Ordering

A set A with a linear order < is well-ordered if every non-empty subset of A has a least element.

Formally: ∀ S ⊆ A, S ≠ ∅ ⟹ ∃ m ∈ S: m ≤ s for all s ∈ S

Examples of Well-Ordered Sets
  • ℕ = {0,1,2,3,…} with the usual ≤ — well-ordered ✅ (every non-empty subset has a smallest element)
  • {0,1,2,…,n} for any finite n — well-ordered ✅
  • {0,1,2,…} ∪ {ω} where ω is bigger than every natural — well-ordered ✅
  • ℤ (integers) with usual ≤ — NOT well-ordered ❌ (the subset {...,-3,-2,-1} has no smallest element)
  • ℝ (reals) with usual ≤ — NOT well-ordered ❌ (e.g. the interval (0,1) has no least element)
  • Any finite set with any linear order — always well-ordered ✅

Step 2 — Order Isomorphism

Definition — Order Isomorphism

Two well-ordered sets (A, <_A) and (B, <_B) are order isomorphic if there is a bijection f: A → B that preserves order:
a₁ <_A a₂ ⟺ f(a₁) <_B f(a₂)

Order isomorphism captures the idea of "same ordering structure" regardless of what the elements are.

Example — Order Isomorphism

{1,2,3} with usual order is isomorphic to {a,b,c} ordered a<b<c via f(1)=a, f(2)=b, f(3)=c.
ℕ and {2,4,6,8,…} (even numbers) are order isomorphic via f(n)=2n — same order type!
ℕ and ℕ ∪ {ω} are NOT order isomorphic — one has a last element (ω), the other doesn't.

Step 3 — Ordinal Numbers

Definition — Ordinal Number

An ordinal number (or ordinal) is the order type of a well-ordered set — a canonical representative of its isomorphism class.

Von Neumann's construction: Each ordinal IS the set of all smaller ordinals:
0 = ∅
1 = {0} = {∅}
2 = {0,1} = {∅,{∅}}
3 = {0,1,2}
n = {0,1,2,…,n−1}
ω = {0,1,2,3,…} = ℕ

Key insight: Every ordinal α = {β | β < α}. An ordinal literally contains all the ordinals smaller than itself. This makes the ordinals naturally well-ordered by the membership relation ∈.

Step 4 — Two Types of Ordinals

Successor Ordinals

An ordinal α is a successor ordinal if α = β + 1 = β ∪ {β} for some ordinal β.
Examples: 1 = 0+1, 2 = 1+1, 3 = 2+1, …, ω+1 = ω∪{ω}, ω+2, ω+3, …
Every successor ordinal has an immediate predecessor.

Limit Ordinals

An ordinal α > 0 is a limit ordinal if it is NOT a successor ordinal — it has no immediate predecessor.
α is a limit ordinal iff α = sup{β | β < α} (it equals the supremum of all smaller ordinals).
Examples: ω, ω·2, ω², ω^ω, ε₀
ω is the FIRST limit ordinal — it has no predecessor ordinal.

The Ordinal Hierarchy
Finite ordinals: 0, 1, 2, 3, 4, … (each is a successor)
First limit: ω = {0,1,2,3,…}
Successors of ω: ω+1, ω+2, ω+3, …
Second limit: ω+ω = ω·2 = {0,1,2,…,ω,ω+1,ω+2,…}
More: ω·2+1, …, ω·3, …, ω·ω = ω²
Higher: ω³, ω⁴, …, ω^ω, ω^{ω^ω}, …
Epsilon-naught: ε₀ = ω^{ω^{ω^{…}}} (first fixed point of α↦ω^α)
Much higher still: ε₁, ε₂, …, Church–Kleene ordinal ω₁^{CK}, …

Ordinal Arithmetic

Adding Ordinals
  • α + 0 = α
  • α + (β+1) = (α + β) + 1 (successor step)
  • α + λ = sup{α + β | β < λ} for limit ordinal λ
  • Warning: ordinal addition is NOT commutative!
    1 + ω = ω (adding 1 before ω doesn't extend past ω)
    ω + 1 > ω (adding 1 after ω gives a new ordinal)
Multiplying Ordinals
  • α · 0 = 0
  • α · (β+1) = α · β + α
  • α · λ = sup{α · β | β < λ} for limit λ
  • 2 · ω = ω (repeating 2 infinitely gets to ω)
  • ω · 2 = ω + ω > ω (two copies of ω)
  • Not commutative: 2 · ω ≠ ω · 2
⚠️ Ordinals vs. Cardinals

Ordinals and cardinals agree on finite numbers (3 means "three" in both senses). For infinite sets, they diverge. The ordinals ω, ω+1, ω+2, …, ω·2, …, ω² all have the SAME cardinality ℵ₀ — they are all countably infinite, just with different orderings. Cardinals measure size; ordinals measure order type.

Transfinite Induction

Big Picture: Ordinary mathematical induction proves P(n) for all n ∈ ℕ using two steps: prove P(0) and prove P(n)→P(n+1). But what about proving P(α) for ALL ordinals α — including limit ordinals like ω where there's no immediate predecessor? Transfinite induction adds a third step to handle exactly this case.

Recall — Ordinary Induction

Standard Mathematical Induction

To prove P(n) for all n ∈ ℕ:
Base case: Prove P(0).
Inductive step: Assume P(n); prove P(n+1).
Limitation: This only reaches every finite n. It never reaches ω or beyond.

Why We Need a Third Step

Suppose we've proved P(0), P(1), P(2), … P(n) for every finite n. Does P(ω) follow?

NOT automatically! ω is a limit ordinal — it has no immediate predecessor. Standard induction only gives us P(n) → P(n+1), which never crosses the gap to ω.

We need a separate argument: "if P(β) holds for ALL β < ω, then P(ω) holds."

The Three Cases

Transfinite Induction

To prove P(α) for all ordinals α, it suffices to prove:

Case 1 — Base: P(0) holds.

Case 2 — Successor step: For every ordinal β, if P(β) then P(β+1).

Case 3 — Limit step: For every limit ordinal λ, if P(β) for all β < λ, then P(λ).

Strong Form (Transfinite Induction Schema)

Equivalently: if P(β) holds for all β < α implies P(α), for every ordinal α, then P(α) holds for all ordinals.

This is the "strong induction" version — the inductive hypothesis is "P holds for ALL smaller ordinals."

Why It Works

Suppose the three cases hold but P(α) fails for some ordinal α. Let α₀ be the smallest ordinal where P fails (exists by well-ordering of ordinals).
— α₀ ≠ 0 (Case 1 covers this)
— α₀ is not a successor ordinal β+1 (because P(β) holds — α₀ was minimal — so Case 2 gives P(β+1))
— α₀ is not a limit ordinal (because P(β) holds for all β < α₀ — minimal — so Case 3 gives P(α₀))
Contradiction! So no failure is possible — P holds everywhere.

Worked Examples

Example 1 — Every ordinal is either 0, a successor, or a limit

Let P(α) = "α is 0, a successor, or a limit ordinal."

Base (α=0): P(0) holds — 0 is explicitly the base case. ✅
Successor (α=β+1): β+1 is a successor ordinal by definition. ✅
Limit (α=λ): λ is a limit ordinal by definition (any ordinal that's not 0 and not a successor). ✅

By transfinite induction, every ordinal α belongs to exactly one of these three categories. □

Example 2 — Defining a function on all ordinals by transfinite recursion

Define f(α) = "the set of all ordinals ≤ α" for all ordinals α.

Base: f(0) = {0}
Successor: f(α+1) = f(α) ∪ {α+1}
Limit: f(λ) = ⋃{f(β) | β < λ}

Transfinite induction guarantees this is well-defined for all ordinals.

Example 3 — Ordinal arithmetic is well-defined

We can define α + β for all ordinals by transfinite induction on β:
α + 0 = α (base)
α + (β+1) = (α + β) + 1 (successor step)
α + λ = sup{α + β | β < λ} (limit step)

Each step is well-defined because we only reference previously defined values.

Example 4 — Every well-ordered set is isomorphic to a unique ordinal

Let P(α) = "every well-ordered set of order type α is isomorphic to the ordinal α."

Base: The empty set has order type 0. ✅
Successor: If every set of type β is iso to β, then any set of type β+1 is {elements of type β} ∪ {one more element} ≅ β+1. ✅
Limit: A set of limit type λ is the union of initial segments of types β < λ, each isomorphic to β by inductive hypothesis. ✅

Therefore every well-ordered set has a unique ordinal representative. □

Transfinite Recursion

Transfinite Recursion Theorem

Given a function G that maps partial functions to values, there is a unique function F defined on all ordinals such that:
F(α) = G(F ↾ α) for all ordinals α

(where F ↾ α is the restriction of F to ordinals below α.)

This is the engine behind ordinal arithmetic, the cumulative hierarchy of sets (V₀, V₁, …), and many other constructions.

Comparison: Ordinary vs. Transfinite Induction
  • Ordinary induction: 2 steps (base + successor). Reaches all finite n.
  • Transfinite induction: 3 steps (base + successor + limit). Reaches ALL ordinals.
  • The limit step is the key addition — it handles "jumps" at limit ordinals like ω, ω², ε₀.
  • Strong transfinite induction: assume P holds for all β < α; prove P(α). One step covers all three cases.
  • Both rely on the well-ordering of their respective domains (ℕ for ordinary, Ord for transfinite).

3 steps of transfinite induction?
1. Base: P(0). 2. Successor: P(β)→P(β+1). 3. Limit: P(β) for all β<λ → P(λ).
Why do we need the limit step?
Limit ordinals like ω have no predecessor — you can't reach them by successor steps alone.
Strong transfinite induction?
Assume P(β) for ALL β < α; prove P(α). Covers base, successor, and limit in one step.
Why does transfinite induction work?
Ordinals are well-ordered — if P fails anywhere, there's a LEAST failure, which leads to a contradiction with one of the three cases.
What is transfinite recursion?
Defining a function F(α) for all ordinals using: base, F(β+1)=G(F(β)), F(λ)=G({F(β)|β<λ}).
Can ordinary induction prove P(ω)?
NO — ordinary induction proves P(n) for all finite n, but cannot cross to ω without the limit step.

📝 Proof Templates

Proof 1 — Every Non-Empty Set of Ordinals Has a Minimum
Claim: Every non-empty class S of ordinals has a least element.
Proof by transfinite induction: Let S be a non-empty class of ordinals. Pick any α ∈ S. Case 1: No ordinal β < α belongs to S. Then α is the least element of S. ✓ Case 2: Some β₀ < α belongs to S. Consider T = S ∩ α = {β ∈ S | β < α}. T is a non-empty set of ordinals, all less than α. α is itself a set, and T ⊆ α. By induction on α (every ordinal is well-ordered as a set by ∈), T has a minimum element m. Since all elements of S less than α are in T, m is the least element of all of S. ✓ Therefore S is well-ordered. □
Proof 2 — Transfinite Induction for Ordinal Addition
Prove: α + 0 = α for all ordinals α.
This follows immediately from the BASE CASE of the recursive definition of ordinal addition: α + 0 := α (definition) Now prove: α + β is defined for all β, by transfinite induction on β. Base (β=0): α + 0 = α. ✓ Successor (β = γ+1): Assume α + γ is defined. Set α + (γ+1) := (α + γ) + 1. ✓ Limit (β = λ): Assume α + γ defined for all γ < λ. Set α + λ := sup{α + γ | γ < λ}. ✓ By transfinite induction, α + β is defined for all ordinals α, β. □
